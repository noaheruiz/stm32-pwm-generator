# stm32-pwm-generator
Generates PWM at 10% duty cycle with 8 bit range.

## Hardware
- Board: STM32 Nucleo-F401RE

## CubeMX
- Timers: TIM2 Channel 1
- Pulse: 25
- Counter Period: 255
- Duty Cycle: 10%
